# 🔐 Metasploitable 2 Vulnerability Assessment

A hands-on Vulnerability Assessment and Penetration Testing (VAPT) project performed on **Metasploitable 2** using **Kali Linux** in a secure lab environment.

## 🚀 Tools Used
- Nmap
- Metasploit
- Nikto
- SMBClient
- MySQL Client
- SSH
- Telnet

## 📌 Key Findings
- Exploited vsftpd 2.3.4 backdoor
- SSH & Telnet login using default credentials
- Anonymous SMB access
- MySQL root access without password
- Multiple Apache web vulnerabilities
- Exposed Tomcat management interface

## 🎯 Skills Demonstrated
- Network Scanning
- Service Enumeration
- Vulnerability Assessment
- Exploitation
- Security Analysis

## ⚠️ Disclaimer
This project was conducted **only in an isolated lab environment** for educational and ethical hacking purposes.
